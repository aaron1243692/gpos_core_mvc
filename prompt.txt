Implement FIFO for product sales and product profit calculation.

Project:
C:\project\gpos_core_mvc

Goal:
Implement First In, First Out FIFO for products so important numbers are accurate:

* Product COGS
* Gross profit
* Net profit reports
* Date range profit
* Product stock deduction

Important:
FIFO applies to products only.
Do not apply FIFO to fuel.
Fuel should use tank weighted average cost later because fuel mixes.

Current product rule:

* Products have batches.
* Each batch can have different cost_price.
* Latest active batch selling_price is the current selling price.
* POS must sell from display_stocks only.
* POS must not sell from warehouse_stocks.

FIFO rule:
When selling a product, deduct from the oldest available display stock batch first.

Oldest batch order:

1. product_batches.created_at ascending
2. product_batches.id ascending if created_at is not reliable

Example:
Candy Batch 1:

* cost_price = 1
* display quantity = 10

Candy Batch 2:

* cost_price = 2
* display quantity = 20

Current selling price from latest active batch:

* selling_price = 3

Customer buys 15 candies.

FIFO result:

* 10 pcs from Batch 1
* 5 pcs from Batch 2

Save product_sales rows:
Row 1:

* batch_id = Batch 1
* quantity = 10
* cost_price = 1
* selling_price = 3
* subtotal = 30
* gross_profit = 20

Row 2:

* batch_id = Batch 2
* quantity = 5
* cost_price = 2
* selling_price = 3
* subtotal = 15
* gross_profit = 5

Total:

* gross sales = 45
* COGS = 20
* gross profit = 25

SubmitSale product logic:
When checkout succeeds:

1. Validate product quantity.
2. Validate total display stock across active batches is enough.
3. Get display_stocks for the product ordered by FIFO.
4. Deduct quantity from oldest batches first.
5. Create one product_sales row per consumed batch.
6. Save cost_price, selling_price, subtotal, and gross_profit in product_sales.
7. Write stock_movements if available.

Required product_sales fields:
Add only if missing:

Use existing fields if already available:

Formula:
subtotal = selling_price * quantity
cogs = cost_price * quantity
gross_profit = subtotal - cogs
